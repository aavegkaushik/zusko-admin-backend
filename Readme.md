# 🧺 Zusko Vendor Dashboard – Backend (Express + MongoDB)

This is the backend service for the **Zusko Vendor Dashboard**, designed exclusively for **local laundry vendors**.  
Vendors can:

- Login securely using JWT
- View all customer orders assigned to them
- Update order statuses (pending → in-progress → ready → completed)
- View real-time analytics (Day, Week, Month)
- Generate PDF invoices for any order
- Access vendor statistics & earnings summary

This backend powers the Vendor side of the Zusko Laundry ecosystem.

---

## 🚀 Tech Stack

### **Backend**
- Node.js + Express.js
- MongoDB + Mongoose
- JWT Authentication
- PDFKit (Invoice Generator)
- DayJS (Date formatting)
- Helmet (Security)
- Express-Rate-Limit
- Morgan (Request Logger)
- dotenv (Environment Configuration)
